Enter the values for these Slack message fields. Either use direct value or a splunk token (i.e. $result.fieldname$) If you use the last case, the field has to be included in the fields list above. For an explanation of these fields, have a look at the Slack API message attachments.

Add additional fields you want to add in the alert. Use JSON notation to define fields and their values. The value can be a direct string or a Splunk token (i.e. $result.fieldname$).
Example:
{ "Field header 1": "Fixed value",
"Field header 2": "$result.fieldname$" }
The value for Field header 2 will be replaced by Splunk.
